# LABORATORIO ADADR 2025

Este archivo contiene 2 carpetas, uno para la parte 1 El Problema de los Generales Bizantinos, y otro para la parte 2 Protocolos de Commit de Transacciones.

Para ejecutar el codigo, primero instalar PyDistSim en un virtual environment de Python 3.11, como lo indica la [documentación](https://pydistsim.readthedocs.io/install/installation.html)

En cada carpeta se encuentran los algoritmos creados para la simulaciones de cada parte, así como una notebook donde se hace la experimentación de dichas simulaciones.

Cada corrida de los algorimtos, se imprimen logs para dejar en claro la acción de cada nodo. Para notar mejor esto, se recomienda ejecutar simulaciones con pocos nodos.
